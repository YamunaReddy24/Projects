AgriLink_Complete - Frontend landing

Unzip and import as a Maven project in Eclipse/IntelliJ.
To run:
1) Ensure Java 17 and Maven are installed.
2) (Optional) Create MySQL database 'agrilinkdb' or update application.properties.
3) Run AgriLinkApplication (src/main/java/com/agrilink/AgriLinkApplication.java) as Spring Boot App.
4) Open http://localhost:9090/

Replace images in src/main/resources/static/images with high-res photos (banner.jpg, fields.jpg, tractor.jpg, green-farm.jpg) for a polished look.
